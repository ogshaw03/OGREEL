# OGREF スマホ版 開発ノート（引き継ぎ用）

このファイルは「スマホ版UI」の検討状況を別チャットへ引き継ぐためのメモです。
本体アプリ（PC/Beta）の仕様変更とは分けて、スマホUIの設計・プロトタイプ作業をここに集約します。

## 対象・前提
- リポジトリ: `ogshaw03/OGREF`（ブランチ: `main`、開発は `claude/loving-franklin-FxIWN`）
- 方針:
  - **機能は PC版（`OGREF_Beta.html`）と同等**。**変えるのは UI のみ**。
  - **見やすさ最優先**、親指操作前提のモバイル最適化。
  - 実装は **レスポンシブ（同一ファイルに mobile レイアウトを追加）** を想定（ロジック流用・機能ズレ防止）。

## 成果物
- **操作プロトタイプ（機能なし・UI/遷移確認用）**: `OGREF_mobile_prototype.html`
  - 公開URL: https://ogshaw03.github.io/OGREF/OGREF_mobile_prototype.html
  - キャッシュ回避は末尾に `?v=数字` を付ける（例: `...html?v=6`）
  - Firebase 不使用・サンプルデータのみ。タップでトースト等のダミー挙動。

## 現在のスマホUI構成（プロトタイプ）
- **上部バー**: ワークスペース名 ▾（切替シート）/ 🔍検索 / **Aa（タイトル表示の切替）** / ⋮メニュー
- **コンテンツ（全幅）**:
  - 上部に **タグ一覧フィルタ**（チップを横に並べ、折り返し表示）
  - **動画グリッド**：**タイル（サムネ）のみ表示が既定**。タイトルは「Aa」ボタンで ON/OFF 切替。
- **下部バー（2段・横スクロール）**:
  - 左端に **追加ボタン（色付き・縦長固定）**
  - 上段：**ライブラリ（すべて/⭐）＋ プラットフォーム（YouTube/Vimeo/ニコニコ/X/Instagram/TikTok/LinkedIn/Drive/ローカル）**
  - 下段：**フォルダ**
- **メニュー（⋮）内**:
  - 💾 ローカルフォルダ（スマホでは利用頻度が低いためメニュー格納で確定）
  - 👥参加者一覧 / 🔗共有 / 🔐アクセス管理 / ⚙設定 / 🚪退出
- **画面遷移**: ログイン → ワークスペース選択 → 一覧 → プレイヤー（全画面・ループ）/ 各シート / 比較（縦2分割・横回転で左右）

## 確定済みの技術メモ
- 画面切替は `visibility/opacity`（transformの覗き・誤タップを回避）。
- 下部固定対策：`html,body{overflow:hidden}` + デバイス高さ `100dvh`（アドレスバーでページごとスクロールする問題を解消）。
- 検索欄の `autofocus` は読込時の自動スクロール不具合の原因だったため削除済み。
- スマホで実機表示するにはローカルHTMLではなく**URL（GitHub Pages）**で開くのが確実。
