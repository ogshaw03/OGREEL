# OGREF リリースノート

## Beta_v0.2.0（今回）

### 📱 モバイル端末に対応
- スマートフォン表示に対応しました。

### 🎨 プラットフォームアイコンを変更
- 各プラットフォーム（YouTube / Vimeo / ニコニコ / X / Instagram / Google Drive / LinkedIn / TikTok / ローカル）のアイコンを、各サービスのロゴに近いデザインへ変更しました。

### 💬 意見箱を追加
- ご意見・ご要望を送れる「意見箱」を追加しました（お名前は任意）。
- 送信した意見には、運営が設定した**ステータス**（要検討／対応中／対応予定 など）と**運営コメント**が表示されます。
- 他の利用者の意見も**匿名**で確認できます（重複の防止・対応状況の共有のため）。
- 運営は運営コンソールの「意見一覧」でステータス設定・コメント・削除が行えます。

### 📣 更新情報（パッチノート）を追加
- アプリ内で更新内容を確認できる「更新情報」を追加しました。
- サイドバー左下（スマホは ⋮ メニュー）から、いつでも開けます。
- アップデートがあったときは、初回に一度だけ自動表示されます。

> ※ 意見箱の利用には Firestore ルールに `feedback` コレクションの許可が必要です（`read, create, update, delete: if request.auth != null`）。

## Beta_v0.1.4

### 💾 ローカル動画の更新方式を改善
- ローカル動画の更新は**ブラウザのリロード**で反映（自動同期）。各ユーザーが自分のフォルダ基準で更新され、**他メンバーの画面には波及しません**。
- エクスプローラーで削除した動画は、リロード時に**自分の画面だけ**から消えます（他メンバーがまだ持っていれば表示は維持）。
- 追加が1回のリロードで反映。他ユーザーの追加時に全タイルが再読み込みされるチラつきも解消。

### 🔐 アクセス管理
- 許可ドメインに該当する**ログイン済みユーザー**をアクセス管理画面に一覧表示。

### ▶ ループ再生
- 右側スライドパネルとモーダルビューアーを**ループ再生**に（ローカル / YouTube / Vimeo）。

### 🎨 UI 改善
- サイドバー最小化アイコンを、サイドバー境界の**小さなタブ**に変更。
- 動画追加モーダルの「取得」ボタンと、ロゴ右の重複ナビボタンを廃止（ヘッダーのホームボタンに集約）。

## Beta_v0.1.0（今回）

`Beta_v0.0.8` 以降の開発（dev）成果をまとめて本番(Beta)へ反映しました。
※ 運営機能（メンテナンス・お知らせ・権限管理）は別アプリ **運営コンソール（OGREF_admin.html）** から操作します。

### 🎬 対応プラットフォームの拡大
- **LinkedIn** に対応（投稿/動画の埋め込み再生）。
- **TikTok** に対応。
- **X（Twitter）/ Instagram / TikTok** をタイルに**サムネ表示**（公式の投稿埋め込み）。
  - 静止画サムネが取得できないため投稿埋め込みを表示。中途半端な見切れを防ぐため**枠内に全体縮小フィット**。
  - 右側スライドパネルでは**縦長表示**。
  - ※自動再生は各プラットフォームの埋め込み仕様に依存。

### 🏷 動画追加の利便性
- URL追加時、**タイトル欄が空なら動画タイトルを自動取得**（YouTube / Vimeo の oEmbed）。手入力した場合はそのまま尊重。

### 💾 ローカル動画
- **`.mov` の再生に対応**（H.264/AAC）。`video/mp4` として読み込むことで再生可能に。
- HEVC/ProRes 等のブラウザ非対応コーデックは、**再生不可の理由を明示**（軽量版に .mp4 を置けば再生可能）。
- フォルダピッカーが**前回開いた場所を記憶**し、動画フォルダ付近から開くように（`id`/`startIn`）。
- ワークスペース入室時にローカルフォルダを**自動再接続**。作成者が接続したフォルダ名を共有し、各メンバーへ**ワンクリック接続**を案内。
- 別途、変換ツールを `tools/`（`OGREF_convert.bat` ほか）に同梱（HEVC等→H.264 .mp4。ffmpeg版とGUI手順）。

### 🖱 操作
- タイルのプレビュー上で**中ボタン（ホイール）スクラブ**。押しながら左右で再生位置を前後に走査（ローカル動画）。

### 👥 メンバー / 権限
- **ワークスペース参加者一覧**（ツールバー「👥 参加者」）。
- 参加者の**ワークスペース退出**（退出すると一覧から削除）。
- **動画追加権限**をオーナーが設定（全員可 / 許可ユーザーのみ / オーナーのみ）。許可ユーザーは参加者画面から指定。
- 権限を **運営 / 管理者 / メンバー** の3区分に整理。
  - **運営**：アクセス権付与・メンテナンス制御・運営コンソール利用が可能。
  - **管理者**：アクセス権付与（OGREF内のアクセス管理画面）が可能。
  - **メンバー**：利用のみ。

### 🛠 運営機能（運営コンソール OGREF_admin.html から操作）
- **メンテナンス**：開始するとアクセス中の全メンバーがメンテナンス画面に。**解除すると全メンバーのページを自動再読み込み**。
- **全メンバーへのお知らせ**：画面上部にバナー表示。各メンバーは**確認後に自分で消去**可能。
- アクセス管理：運営/管理者/メンバーの追加・変更・削除。最後の運営ユーザーは降格・削除不可（ロック防止）。

### 🎨 その他
- ヘッダーロゴのサイズ調整（90%）。
